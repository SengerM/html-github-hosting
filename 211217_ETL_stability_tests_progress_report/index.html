In this document I provide a brief update on the progress of the stability tests of the ETL detectors. Previous report can be found in reference .

Contents

Introduction

The ETL is a new sub-detector to be installed in the CMS detector during the long shutdown 3 . It is going to be implemented with LGAD detectors, a relatively new technology in silicon. Stability tests are ongoing, described in this document, to verify the long-term performance of this technology. A first description of the tests and the planned setup can be found in .

Current status of the setup

The setup, described in , is still under commissioning. The construction of the mechanical system to move the beta source and the timing reference is the most complicated part as it has to be custom-made, but it is almost completed. Some pictures of the setup can be found in where we can see the mentioned mechanical system partially assembled.

Pictures of the setup.

All the parts for the setup are already available in our lab with the exception of the MCP-PMT timing reference  which should arrive within the following two weeks.

Current results

Despite the setup not being ready yet, some stability tests were already performed on irradiated devices. These tests consisted on keeping the devices biased at operating voltage"Operating voltage" means that bias voltage such that 10 fC of charge was produced by the detectors according to a plot sent to me by Nicolò. for "a long period of time" while monitoring the bias voltage and current. In it shown the bias current, bias voltage, ambient temperature and ambient humidity for eight devices tested for a period of about 4 months. As the setup is still in commissioning the test was interrupted for small periods of time to perform technical tasks. This can be seen in the plots in the periods were the bias voltage was reduced or the temperature increased. Within the periods in which the bias voltage and the temperature were at correct values, with extensions up to 2.5 weeks, no instabilities were observed in the bias currentThis means that there was no point in time at which the bias current showed a peak, or any strange behavior, while both bias voltage and temperature were at the proper values..

An exception is device #10, W18 T4 4e14 neq cm-2 which reached breakdown before the expected bias voltage. This device, however, showed a different behavior even in the initial IV curves measured in the probe station, see (taken from plot in Figure 7 of reference ).

IV curve of device #10, W18 T4 4e14 neq cm-2.

A more detailed analysis of this data is pending and will be performed once the setup is fully operational, which should happen soon.

Conclusions

  • Activities are ongoing according to the plan.
  • Setup is almost complete, expecting to have it fully operational in one month.
  • Current tests (without beta source) show no instabilities on the irradiated LGADs.
